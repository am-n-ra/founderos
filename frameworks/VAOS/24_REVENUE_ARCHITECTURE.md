# VAOS — 24 REVENUE ARCHITECTURE

## Purpose

How does money enter KORA?

## Revenue Streams

| Stream | Description | Customer | Price | Priority |
|--------|-------------|----------|-------|----------|
| API payante | Inference as a Service | Startups, devs, entreprises | $0.001-0.01/request | Primary |
| Enterprise licensing | Modele prive + support | Gouvernements, grandes entreprises | $5K-50K/an | High |
| Grants | Recherche, impact | Fondations (HAIDI, AI4D, etc.) | $10K-100K | Critical now |
| EA Trading | Trading automatise | — | $20-200/mo | Oxygen |
| Consulting/Formation | Expertise IA | Entreprises, institutions | $1K-5K/mission | Medium |
| API freemium | Gratuit pour usage limite | Particuliers, etudiants | $0 | Acquisition |

## Primary Revenue Model

**Model:** API payante (inference) + Enterprise licensing

**Why:** 
- L'API est le standard de l'industrie (comme OpenAI)
- Enterprise = contrats longs, revenue stable
- Freemium pour la mission (acces gratuit pour les non-instruits)

**Unit economics (estimation):**
- Cout d'inference par requete: ~$0.0001 (optimise sur T4)
- Prix par requete: $0.001
- Marge brute: 90%
- CAC: $0 (organic, community)
- LTV: $100-500/client enterprise

## Revenue Sequence

1. **Maintenant:** EA Trading ($0-200/mo) — oxygen
2. **J-19:** HAIDI ($38K) — si accepte, 12 mois de runway
3. **Mois 3:** API payante — premiers dollars recurring
4. **Mois 6:** Enterprise — contrats longs
5. **An 2:** Grants + API scale — $5K-20K/mo

## Revenue Risks

- **Si EA Trading echoue** → perte de l'oxygen, mais pas fatal si HAIDI passe
- **Si HAIDI rate** → il faut soit un autre grant, soit API revenue rapide
- **Si API revenue < $100/mo a 6 mois** → modele economique a revoir
