# VAOS — 16 ECOSYSTEM MAP

## Purpose

How do all ventures under the founder mission relate to each other?

## Founder Mission

> Accelerer le rattrapage technologique, economique et humain de l'Afrique afin que le lieu de naissance ne determine plus les opportunites.

## Venture Map

```
Founder Mission (Kheir)
    │
    ├── KORA (IA Souveraine)
    │       │
    │       ├── alimente OMNI en capacites IA
    │       └── cree les actifs pour EDEN VALLEY
    │
    ├── OMNI (Infrastructure numerique)
    │       │
    │       ├── MVP existe (Djanta Tech Hub soumis)
    │       └── peut integrer les modeles KORA
    │
    └── EDEN VALLEY (Innovation ecosystem)
            │
            ├── Venture studio sur les actifs KORA + OMNI
            ├── Incubation de startups africaines
            └── Academie pour former les talents
```

## Venture Roles

| Venture | Contribution | Revenue Model | Lever |
|---------|-------------|---------------|-------|
| KORA | Modeles IA souverains | API, licensing, enterprise | IA Souveraine |
| OMNI | Infrastructure logistique | Commission, SaaS | Innovation |
| EDEN VALLEY | Creation d'entreprises | Equity, incubation fees | Innovation |

## Dependencies

- KORA → besoin de compute (ST Digital, Kaggle)
- OMNI → peut fonctionner independamment (MVP existe)
- EDEN VALLEY → besoin de KORA ou OMNI comme actif de base

## Sequence

1. **KORA** premier (en cours)
2. **OMNI** en parallele (deja commence)
3. **EDEN VALLEY** quand KORA ou OMNI a un actif tangible

## Ecosystem North Star

Impact cumule des 3 ventures sur le rattrapage africain.
