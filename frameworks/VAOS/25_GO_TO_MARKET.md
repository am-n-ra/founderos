# VAOS — 25 GO-TO-MARKET

## Purpose

How do we enter the market?

## Market Segmentation

| Segment | Size | Readiness | Competition | Entry |

## Beachhead

**Beachhead:** Developpeurs africains (instruits, utilisent deja l'IA)

**Why:**
- Deja familiers avec l'IA — zero education necessaire
- Ont un smartphone, une connexion — zero friction
- Peuvent tester l'API en 5 minutes — adoption rapide
- Deviennent des ambassadeurs pour le reste du marche

## GTM Sequence

1. **Devs africains** — API gratuite, communaute, open source
2. **Startups** — API payante, pricing local
3. **Universites** — Gratuit pour la recherche, forment la prochaine generation
4. **Gouvernements** — Souverainete, data locality
5. **Non-instruits** — Interface vocale, partenaires distribution
