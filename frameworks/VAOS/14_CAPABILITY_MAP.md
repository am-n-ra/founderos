## Critical Capabilities

1. **Training Engineering** — Train a model from scratch on limited compute. Make-or-break.
2. **Data Engineering** — Collect, clean, structure training data. Most underestimated bottleneck.
3. **Distribution** — Reach users. Without it, the best model is invisible.

## Gap Analysis

- **Training:** Never done it. First model will be a learning curve.
- **Data:** No pipeline. Start by scraping public sources (Ewe Bible, web, Masakhane).
- **Distribution:** Zero. Start with close circle (devs, students).
