# DAILY PLAN

## Model

DAILY_PLAN is a template. The actual daily plan is written to `State/DAILY_PLAN.md` by `kickoff.py` at boot or explicit kickoff.

This file defines the structure. Do not edit this file for daily changes — edit `State/DAILY_PLAN.md`.

---

## Validation

A valid DAILY_PLAN must:
1. Have at least 1 actionable item (not "wait" or "plan")
2. If cash < 1,500: have at least 1 revenue-generating action as #1
3. Have no empty time blocks (each block either has an action or is marked REST)
4. Be timestamped with current date
