# VAOS — 13 NORTH STAR

## Purpose

The single metric that measures success.

## The Metric

**North Star:** Nombre d'Africains utilisant activement des modeles KORA entrainees depuis zero.

## Why This Metric

- Combine performance (ils utilisent parce que c'est bon) et mission (ils utilisent parce que c'est accessible)
- Leading indicator du rattrapage technologique — chaque utilisateur de moins importe
- Mesurable: nombre d'appels API, d'utilisateurs actifs, de voix sessions
- Aligne toute l'equipe

## What It Predicts

Si ce nombre monte :
- Les modeles sont suffisamment performants (sinon personne n'utiliserait)
- La distribution fonctionne (sinon personne n'aurait acces)
- Le compute tient la charge
- La mission avance

## What It Does NOT Measure

- Le revenue (important mais pas le nord)
- Les benchmarks bruts (utile mais intermediaire)
- Les papiers publies (important mais pas le nord)

## How to Move It

- Entrainer de meilleurs modeles → retention utilisateur
- Interface vocale en langues africaines → nouveau public
- Distribution via smartphones → reach
- API abordable → adoption developpeurs
- Partenariats institutions → acces rural

## Annual Target

**12 mois:** 10,000 utilisateurs actifs mensuels (dont 50% via voix, 50% via texte).
