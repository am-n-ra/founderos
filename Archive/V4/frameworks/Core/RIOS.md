# RIOS — Research & Intelligence Operating System
Last Updated: 2026-06-27

## Purpose
Systematic external research pipeline. Turns questions into structured intelligence.

## Sources
- Web search (via web search tool / API)
- Watchtower (automated hourly/daily checks)
- OOOS (opportunity scoring results)
- Manual research (user-directed deep dives)

## Research Pipeline
1. **Question** — What do we need to know?
2. **Source selection** — Web, watchtower, expert, document
3. **Collection** — Gather raw data
4. **Analysis** — Extract insights, patterns
5. **Storage** — Write to KNOWLEDGE.md, TIMELINE.md, or project KNOWLEDGE.md

## Integration
- **OOOS**: Feeds opportunity scoring with fresh data
- **LEOS**: Learning research feeds into capability gaps
- **MOS**: Competitive intelligence feeds mission strategy
- **DAOS**: Daily research actions
