# Lean Canvas — FounderHQ

| Section | Contenu |
|---------|---------|
| **Problème** | Difficulté d'accès aux produits essentiels (santé, énergie) en Afrique francophone. Manque de distribution structurée. |
| **Solution** | Écosystème de produits digitaux et physiques. Distribution via TikTok Shop + WhatsApp + site e-commerce. Automation via FounderOS. |
| **Proposition de Valeur Unique** | "Des produits qui changent la vie, distribués avec la puissance de la tech, accessibles à tous en Afrique." |
| **Avantage Concurrentiel** | FounderOS comme avantage opérationnel. Agilité extrême. Pas de levée de fonds -> pas de dilution, pas de pression des investisseurs. |
| **Canaux** | TikTok (vidéo organique + TikTok Shop), WhatsApp (vente directe + service client), Site e-commerce (à construire). |
| **Segments Clients** | Jeunes adultes (18-35) africains connectés. Consommateurs urbains et péri-urbains en Afrique francophone. |
| **Structure de Coûts** | Production vidéo, achat stock, data, hébergement. ~200K FCFA/mois minimum vital. |
| **Flux de Revenus** | Vente produits (marge 30-50%), commissions affiliation, abonnements (future). |
| **Indicateurs Clés** | Vues TikTok → Clics boutique → Ventes → MRR. CAC, LTV, taux de conversion. |
