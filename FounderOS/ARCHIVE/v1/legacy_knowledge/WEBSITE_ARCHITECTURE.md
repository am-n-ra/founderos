# Website Architecture

## Structure Proposée (FounderHQ.com)
1. **Accueil** — Manifeste + Produits phares + FounderOS (OS qui fait tourner tout ça)
2. **Produits** — Grid des 4 produits. Chacun = fiche dédiée avec description, prix, TikTok embed, WhatsApp CTA
3. **À Propos** — Vision 2028 + Mission 2026 + Parcours founder
4. **Blog/Notes** — Réflexions, apprentissages, building in public
5. **Contact** — WhatsApp direct + formulaire

## Stack Suggérée
Next.js + Tailwind + Hosting pas cher (Vercel ou Netlify)
Domaine : founderhq.com

## Priority
MVP = landing page statique avec les 4 produits. Vidéos TikTok embed. WhatsApp CTA. Rien de plus.
