# WORKFLOW_REGISTRY V1

## Purpose
FounderOS does not execute tasks. FounderOS executes workflows.

## Prime Directive
Never execute directly. Always identify the workflow first.

---

## WORKFLOW: AI_VIDEO

Objective: Produce consistent, scalable AI-generated visual content (any type).
Required Outcome: Final video assets.

Dependency Tree:
Content Type Selection → Creative Strategy → Story Bible → Script → Entity Registry → Entity Sheets → Environment Sheets → Action Sheets → Scene Sheets → Shot Sheets → Start Frames → End Frames → Video Prompts → Generation → Assembly → Publishing Assets.

Story Bible defines: mission, audience, visual identity (color palette, lighting, camera style), narrative identity (tone, emotion, narration), brand identity (values, voice), recurring entities, recurring locations, content rules, forbidden elements.

Mandatory Assets:
- Based on content type:
  - ADVERTISEMENT: OFFER.md, CUSTOMER_PROFILE.md, PRODUCT_SHEET.md, CTA
  - DOCUMENTARY: RESEARCH_SOURCES.md, FACT_VALIDATION.md, NARRATION_PLAN.md
  - PODCAST: HOST_SHEET.md, GUEST_SHEET.md, STUDIO_SHEET.md, OUTLINE.md
  - EDUCATIONAL: LEARNING_OBJECTIVE.md, TEACHING_PLAN.md
  - STORYTELLING: NARRATIVE_ARC.md, CONFLICT.md, RESOLUTION.md
  - FOUNDER_JOURNAL: FOUNDER_SHEET.md, CHALLENGE.md, LESSON.md
- Universal: ENTITY_REGISTRY.md, ENTITY_SHEETS, ENVIRONMENT_SHEETS, ACTION_SHEETS, SCENE_SHEETS, SHOT_SHEETS, START_FRAMES, END_FRAMES, VIDEO_PROMPTS

Quality Gates:
Content type identified ✓ → Type-specific requirements met ✓ → Entity registry exists ✓ → All entity sheets complete ✓ → Action sheets exist ✓ → Scene sheets exist ✓ → Shot sheets exist ✓ → Start frames exist ✓ → End frames exist ✓ → Only then: Video Prompts.

Forbidden: Script → Video Prompt. Prompts before entity sheets.

## Rules

Workflow Discovery: If workflow exists, use it. If not, create, store, register, reuse.

Workflow Improvement: After every execution, ask: can this be improved? If yes, update.

Workflow Completion: Complete only when all required assets exist, all quality gates pass, required outputs exist, completion criteria satisfied.

## FounderOS Rule
Never think in tasks. Think in workflows. Never think in prompts. Think in production pipelines. Never think in outputs. Think in systems.
