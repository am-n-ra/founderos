# FounderOS Directory Standard

FounderOS/
  SYSTEM/       - OS files (read-only specs)
  STATE/        - Current reality (profiles, twin, priorities)
    SNAPSHOTS/  - Daily state backups
  PROJECTS/     - Each venture: ProjectName/
    ProjectName/
      Project_OS/  - Profile, State, Roadmap, Backlog, Decisions
      Code/        - Git repos
      Docs/
      Research/
      Reports/
      Assets/
      Strategy/
  KNOWLEDGE/    - Research, Learning, Frameworks, Decisions, Insights
  REPORTS/      - Daily/, Weekly/, Monthly/, Quarterly/
  AUTOMATIONS/  - Agents/, Workflows/, Scripts/, Logs/
  ARCHIVE/      - Dead projects, old specs
  BACKUPS/      - FounderOS snapshots
