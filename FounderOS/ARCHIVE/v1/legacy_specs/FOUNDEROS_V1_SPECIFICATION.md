# FounderOS v1 Specification

## Master Document

### Vision
Un systeme qui permet a un founder de poursuivre une mission sur 20+ ans sans perdre le contexte, les decisions, les projets, ni la memoire, independamment du LLM, de l'IDE ou de la machine.

### Philosophy
- Tout ce qui compte doit etre un fichier
- Le systeme survit si OpenAI/Anthropic/Google disparaissent
- Le founder n'explique jamais qui il est plus d'une fois
- Chaque jour commence ou le precedent s'est arrete

### Architecture
- PSOS (Constitution) -> MOS (Orchestrateur) -> OS specialises + Engines
- KERNEL au demarrage, RUNTIME pendant la journee, STATE_ENGINE pour la continuite
- TOS pour le temps, POE pour l'humain, OPERATING_MODES pour le contexte

### Stack Complete
SYSTEM/: PSOS, MOS, DAOS, SOS, AOS, KMOS, PMOS, RIOS, FAOS, CEOS, LEOS, GOS, DTOS, EOS, AAOS, MAOS, CAOS, VEAOS, ASTRA, RUNTIME, KERNEL, TOS, STATE_ENGINE, OPERATING_MODES_ENGINE, POE, ARE, EVENT_ENGINE, MSE, IMPLEMENTATION_SPEC, FILE_TEMPLATES
