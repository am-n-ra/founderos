# FounderOS Bootstrap Package

## Installation en 5 minutes

### 1. Creer le dossier
```
mkdir FounderHQ
cd FounderHQ
```

### 2. Copier FounderOS
Copier le dossier FounderOS/ complet dans FounderHQ/

### 3. Ouvrir OpenCode
```
opencode
```
dans FounderHQ/

### 4. Demarrer
Taper:
```
boot
```

Ou copier-coller le prompt de boot:
```
Je suis sous FounderOS. Lis ces fichiers et fais moi un briefing:

- STATE/CURRENT_STATE.md
- STATE/ACTIVE_PRIORITIES.md
- STATE/OPERATING_MODE.md
- STATE/DIGITAL_TWIN.md
- STATE/MISSION_PROFILE.md
- STATE/PROJECT_REGISTRY.md

Puis reponds:
"Bonjour. Mode [X]. Priorite: [Y]. Voici la situation et ce que je recommande aujourd'hui."
```

### 5. Genesis (premiere fois)
Si les fichiers STATE/ n'existent pas:
- L'agent va creer FOUNDER_PROFILE.md, MISSION_PROFILE.md, CURRENT_STATE.md
- Tu reponds a ~10 questions
- Le systeme est pret

### Compatible
OpenCode, Cursor, Windsurf, Claude Code, VS Code + AI
