# OMNI — L'Index du Commerce de Proximité

**Status:** `VALIDATION`

## Mission

Indexer l'économie de proximité africaine sur une carte interactive. L'acheteur trouve en un clic tout ce qui existe autour de lui. Le commerçant devient visible pour la première fois.
