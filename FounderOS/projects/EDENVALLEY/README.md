# EDENVALLEY — Innovation Academy & Incubator

**Status:** `IDEA`

## Mission

Collect innovations enabled by KORA's frontier models and guide them toward venture creation. An academy where even the most disadvantaged — those who cannot read, write, or speak a foreign language — can build software and create ventures using KORA's models in their own language.
