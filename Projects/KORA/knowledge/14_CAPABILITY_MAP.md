# KORA LAB — 14. CAPABILITY MAP

**Version:** 1.0

---

# Purpose

The Capability Map defines every organizational capability Kora Lab must progressively develop to fulfill its mission.

Strategic Assets are what Kora owns.

Capabilities are what Kora can repeatedly do.

Assets create advantage.

Capabilities create assets.

Without capabilities,

strategic assets cannot exist.

---

# What Is A Capability?

A capability is the organization's repeatable ability to execute a complex activity at a high level.

Unlike projects,

capabilities remain after projects end.

Unlike products,

capabilities generate many future products.

Capabilities compound.

---

# Decision Filter

Before launching any initiative ask:

1. Which capability does this strengthen?
2. Is this capability foundational or secondary?
3. Does it create future strategic assets?
4. Can this capability be reused across multiple projects?
5. Will strengthening this capability still matter in ten years?

If the answer is "No" to most questions,

the initiative probably creates activity,

not capability.
