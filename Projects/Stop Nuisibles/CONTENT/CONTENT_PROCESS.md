# Processus de Création de Contenu — Stop Nuisibles

## Ordre d'exécution
1. **Product Sheet** → source de vérité unique
2. **Thumbnail / Image produit** → NanoBanana Pro (9:16 pour shorts)
3. **Vidéo** → Veo 3.1 (utilise l'image comme référence)
4. **Voix off** → TTS
5. **Montage** → combiné avec le texte

## Règles
- Toujours vérifier le format avant de générer
- 9:16 pour TikTok/Insta Reels/Meta (shorts)
- 1:1 pour carousels Instagram
- 16:9 uniquement pour YouTube horizontal (si applicable)
- La Product Sheet est l'unique source de vérité — tout contenu doit être cohérent avec elle
